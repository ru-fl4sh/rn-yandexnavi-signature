# rn-yandexnavi-signature

React Native library for signing yandexnavi url

## Installation

```sh
npm install rn-yandexnavi-signature
```

## Usage

```js
import { multiply } from "rn-yandexnavi-signature";

// ...

const result = await multiply(3, 7);
```
